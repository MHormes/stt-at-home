# STT at Home

Self-hosted speech-to-text for local network use. Record audio from any device on your LAN, get a transcription back instantly.

Built with [whisper.cpp](https://github.com/ggerganov/whisper.cpp) for transcription, a FastAPI backend, and a minimal browser UI served over HTTPS — so microphone access works on mobile without any app install.

All audio stays on your own hardware. Nothing leaves your network.
